# Skyline Surge

A small arcade runner-shooter web game I've been working on.

## How to play
- **A / D**: Move left and right
- **W / Space**: Jump
- **J / K**: Shoot
- **Shift**: Overdrive (when your meter is full)
- **P**: Pause

## Setup
Just open `index.html` in your browser. No local server or build steps needed.

---
Built with HTML5 Canvas and vanilla JavaScript.
